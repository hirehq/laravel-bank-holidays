# Contributing

Thanks for contributing to Laravel Bank Holidays.

## Local Setup

```bash
composer install
composer test
composer lint
```

## Development Guidelines

- Target supported Laravel and PHP versions declared in `composer.json`.
- Add or update tests for every behavior change.
- Keep public API additions documented in `README.md`.
- Run `composer format` before submitting larger changes.

## Pull Requests

- Use a clear title and explain the reason for the change.
- Include tests for fixes and new features.
- Update `CHANGELOG.md` when the change affects package consumers.

## Reporting Issues

When reporting a bug, include:

- Laravel version
- PHP version
- Package version
- Minimal reproduction steps
- Expected behavior and actual behavior
